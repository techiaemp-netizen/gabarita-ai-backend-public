# Gabarita AI Backend

Backend público do sistema Gabarita AI - Plataforma de simulados para concursos públicos.

## Tecnologias
- Flask
- Python 3.11
- Vercel (Deploy)

## Endpoints
- `/api/opcoes/blocos-cargos` - Lista blocos e cargos disponíveis
- `/api/opcoes/cargos/{bloco}` - Lista cargos por bloco
